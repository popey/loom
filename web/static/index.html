Quick Usage Examples

Via CLI:

# Export everything
loomctl export --output backup.json

# Export specific groups
loomctl export --include core,workflow --output partial.json

# Import with merge strategy
loomctl import backup.json

# Dry run before importing
loomctl import backup.json --dry-run
What Gets Exported?
  • Core (7 tables): providers, projects, org_charts, org_chart_positions, agents, credentials
  • Workflow (5 tables): workflows, workflow_nodes, workflow_edges, workflow_executions, workflow_execution_history
  • Activity (7 tables): users, activity_feed, notifications, notification_preferences, bead_comments, comment_mentions, conversation_contexts
  • Tracking (4 tables): motivations, motivation_triggers, milestones, lessons
  • Logging (3 tables): logs, request_logs, command_logs
  • Analytics (2 tables): usage_patterns, optimizations
  • Config (2 tables): config_kv, distributed_locks (PostgreSQL), instances (PostgreSQL)

Total: 30 database tables
